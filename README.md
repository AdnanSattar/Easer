簡介
-----
在不同的狀況（Event）下載入指定的情境（Profile）。每個情境（Profile）由多個動作（Operation）組成。

所有Event以樹狀鏈接，會以後序遍歷選擇Profile載入。

如有興趣參與開發，請看[TODO](TODO)。

已支持功能
----------
### Event
* 日期
* 時間
* WiFi連接名稱
* 位置（基於基站）

### Operation
* WiFi開關
* 數據網絡開關
* 藍牙開關
* 自動旋轉開關
* 發送Broadcast
* 控制亮度
